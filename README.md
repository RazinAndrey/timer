# ⏱️ Timer

Таймер на Angular + Electron

## Основные команды

### Запустить Angular в браузере

```bash
npm start
```

### Собрать Angular и запустить в Electron

```bash
npm run electron-build
```

### Упаковка приложения

```bash
npm run make
```
